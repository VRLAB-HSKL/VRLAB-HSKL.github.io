Unity im VR-Labor

Wir verwenden die folgenden Versionen im VR-Labor:

  • 2019.4.11f1 LTS
  • 2020.1.6f1

Dabei verwenden wir prinzipiell, wenn nicht unbedingt notwendig, immer die aktuellste LTS-Version!

Installierte Module

VR

Bei der Installation der Unity-Versionen installieren wir mindestens die folgenden Module:

  • Windows Build Support
  • Android Build Support
  • WebGL Build Support
  • Documentation

Frameworks für die Software-Entwicklung in VR

Für die Entwicklung von VR-Anwendungen verwenden wir hauptsächlich das Framework "Vive Input Utility" (VIU), das in GitHub frei verfügbar ist. VIU bietet die Möglichkeit einen Simulator zu verwenden und kann sowohl Software für eine HTC Vive, eine HTC Focus Plus, eine Oculus Rift, aber auch für Oculus Go und für Google Cardboard-Geräte erzeugen. Für die Entwicklung von Software für die HTC Focus Plus ist auch das Plugin Wave3D erforderlich.

Neben VIU sind auf den Workstations die folgenden VR-Softwarepakete installiert und verwendbar:

  • VRPN
  • MiddleVR 1.7
  • VTK 8.x mit OpenVR

Es gibt eine Installation von Unreal auf den Workstations, die aber vermutlich ein Update erfordert und aktuell nicht aktiv genutzt wird.
